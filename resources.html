GitHub

CodeUp Manchester has a public GitHub repository where we encourage you to store any public projects you work on at CodeUp. If you're not familiar with Git, it's industry-leading version control software and is incredibly popular with professionals and hobbyists alike.

Slack

CodeUp Manchester has a Slack workspace where you can chat, share files, ask questions and get help with problems in between Meetups. Follow the link to get an invitation.
